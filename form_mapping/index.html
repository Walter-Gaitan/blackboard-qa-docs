    Page 1: Course Info

    1. Course Code (Text) -> Map to course_id (e.g., "Tracy")
    2. Change status to 'UNDER SQA' (Radio) -> Action: Select "Done"

    Page 2: Orientation & Modules

    1. Module Visibility (Radio)
    2. "Make sure the following Modules are visible to students..."
    3. Student/Program Resources (Radio)
    4. "Verify the 'Student Resources' (PBSN) / 'Program Resources' (A&S) module is visible..."
    5. Start Here Module Instructions (Radio)
    6. "Check that the 'Start Here' Module has the correct student instructions..."
    7. Course Outline (MSN) (Radio)
    8. "For MSN courses, a course outline and schedule document is provided..."

    Page 3: Syllabus & Banner

    1. Banner Image (Radio)
    2. "Make sure the banner image matches the course code and title."
    3. Syllabus Resource (Radio)
    4. "In the 'Start Course Orientation' module click the resource labeled as 'Course Overview and Syllabus'..."
    5. Header Info (TDAC) (Radio)
    6. "Verify that the header contains the correct class code and name..."
    7. Syllabus Links (Radio)
    8. "Review and make sure all links in the syllabus work correctly."
    9. Comments/Revisions (Radio)
      • "Comments and revisions are hidden."
    10. Track Changes Colors (Radio)
      • "There are no colors reflecting changes."
    11. Syllabus Format (Radio)
      • "The syllabus has a good format..."
    12. TDAC Resources (Radio)
      • "For tDAc courses only, the 'Syllabus part 2' and 'Setting up for success' resources are visible."
    13. Orientation Acknowledgement (Radio)
      • "Verify the 'Course Orientation Acknowledgement' test completion releases all modules..."

    Page 4: Links

    1. Module Release Conditions (Radio)
      • "All Module Release Conditions are correctly set up..."
    2. Raw Links (Radio)
      • "Make sure there are no raw links placed in any Blackboard page. Except for ATI resources."
      • Note: Automation may need to select "Raw links were found" if "No raw links" is unclickable.
    3. Learning Materials Links (Radio)
      • "Make sure all links in the 'Learning Materials' module work. This includes video links and LTI links."

    Page 5: Assignments

    1. Has Assignments? (Radio)
      • "Does the course have assignments?" -> Action: Select "Yes"
      • New Questions (if Yes):
          1. "Banner matches the activity type."
          1. "Instructions have clear guidelines..."
          1. "All links and attachments work..."
          1. "All rubrics are visible..."

    Page 6: Discussions

    1. Has Discussions? (Radio)
      • "Does the course have discussions?" -> Action: Select "Yes"
      • New Questions (if Yes):
          1. "Banner matches the activity type."
          1. "Instructions have clear guidelines..."
          1. "All links and attachments work..."
          1. "All rubrics are visible..."

    Page 7: Tests

    1. Has Tests? (Radio)
      • "Does the course have tests?" -> Action: Select "Yes"
      • New Questions (if Yes):
          1. "Banner matches the activity type."
          1. "The information is correctly displaced in the first page."
          1. "Guidelines match the panel."

    Page 8: Journals

    1. Has Journals? (Radio)
      • "Does the course have Journals?" -> Action: Select "Yes"
      • New Questions (if Yes):
          1. "Banner matches the activity type."
          1. "Instructions have clear guidelines..."
          1. "All links and attachments work..."
          1. "All rubrics are visible..."

    Page 9: Blackboard Tabs

    1. Discussions Tab Order (Radio)
      • "Discussions follow a sequential order (e.g., Module 1 -> Module 15)."
    2. Gradebook Order (Radio)
      • "Gradebook items are organized chronologically or sequentially (Module 1 -> Module 15)."
    3. Groups Tab (Radio)
      • "Groups are set up as indicated above."

    Page 10: Wrap-up

    1. Tracking List Update (Radio)
      • "Add your findings to the Microsoft tracking list." -> Action: Select "Done"
    2. Time Spent (Text)
      • "Add time spent on QA..." -> Action: Enter time string (e.g., "QA time: 15 mins")
    3. Status Change (Radio)
      • "Change the Microsoft tracking list status to 'SQA findings'" -> Action: Select "Done."
